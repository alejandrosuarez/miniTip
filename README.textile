h1. miniTip jQuery Plugin (__written in CoffeeScript__)

The Tooltip jQuery Plugin you've always been looking for.

miniTip is very basic but powerful tooltip jQuery plugin that allows you to show tooltips on a page at any time and it just Works!

If you have any question of find a bug, please raise it <a href="https://github.com/MatthieuA/Mini-Notification/issues">at GitHub</a> or <a href="http://www.twitter.com/matthieu_tweets">on Twitter</a>.
